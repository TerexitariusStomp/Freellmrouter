# Hermes Free Router

A production-ready Hermes Agent model router that decides which LLM provider/model to use from a pool of free or trial-credit LLM API providers.
## Features
- Multi-Provider Support
- Intelligent Routing
- Hermes Agent Compatible
- Quota Tracking
- Health Monitoring

## Note
Portkey.ai is a great alternative to this router codebase for those looking for a managed solution with similar capabilities.
